### to create a project
`django-admin startproject <my_project>`

### to create an app inseide a project
`cd my_project`
`python manage.py startapp <my_app> `


### How to run the projects

`cd firstProject`
`python manage.py runserver`

`cd templatesDemo`
`python manage.py runserver`
